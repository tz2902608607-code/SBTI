# SBTI
SBTI全称Silly Big Personality Test，直译“傻乎乎的大人格测试”，也被网友调侃为“傻大个测试”，是诞生于中文互联网的恶搞类人格测评工具，并非专业心理学测试，2026年4月通过社交平台快速走红，成为现象级网络趣味测试。
该测试源于中文互联网整活文化，由B站创作者推广出圈，凭借贴合日常、自带吐槽感的内容，快速在微博、抖音、小红书等平台刷屏，凭借极强的网感和共鸣度，成为年轻人社交互动、自我调侃的新方式。
- 测试流程：常规题库为31道题，触发特定选项后会新增1道隐藏题目，全程答题耗时短，操作简单。
- - 测评逻辑：从自我、情感、态度、行动驱力、社交五大模型、15个维度进行评分，最终匹配27种人格结果，包含25种常规人格与2种特殊隐藏人格。
  - - 内容风格：摒弃专业心理学术语，题目围绕拖延、内耗、社交反应、日常行为等生活化场景展开，结果标签直白接地气，充满自嘲与趣味感，精准贴合年轻人互联网精神状态。主打趣味社交、自我调侃，适合朋友间互动分享、社交破冰，不具备专业心理诊断价值，与经典的MBTI测试相比，更聚焦当下日常行为，语言更犀利、更具网感，是互联网情绪表达的一种载体。
